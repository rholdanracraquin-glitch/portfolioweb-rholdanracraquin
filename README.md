# Portfolio Website

## 📖 Project Description

This is a modern and responsive personal portfolio website created using *HTML5*, *CSS3*, and *Vanilla JavaScript*. It showcases my profile, skills, projects, and contact information in a clean and professional layout. The website is fully responsive and works on desktop, tablet, and mobile devices.

---

## ✨ Features

- Responsive design
- Sticky navigation bar
- Mobile hamburger menu
- Smooth scrolling
- Active navigation highlighting
- Hero section with typing animation
- About Me section
- Skills section with progress bars
- Projects section
- Contact form with JavaScript validation
- Dark/Light mode toggle
- Back-to-top button
- Modern UI with CSS Grid and Flexbox
